# 🧠 Swaroop's Text Summarizer

An interactive **NLP-powered text summarization web app** that converts long text into concise summaries using classical Natural Language Processing algorithms.

🔗 **Live Demo**
https://swaroop-text-summarizer.vercel.app/

---

# 🚀 Features

✨ **Multiple Summarization Algorithms**

* Frequency Based Summarization
* TextRank Graph Algorithm

📊 **Algorithm Comparison**
Compare summaries generated by different algorithms.

📝 **Flexible Output Modes**

* Paragraph summary
* Bullet point summary

🔍 **Sentence Highlighting**
Shows which sentences were selected from the original text.

🏷 **Keyword Extraction**
Automatically extracts important keywords from the input text.

📂 **File Upload Support**

* TXT
* PDF
* DOCX
* Images (OCR)

📑 **Export Options**

* Download summary as **TXT**
* Download summary as **PDF**

🔗 **Shareable Summary Links**
Generate a link to share your summary.

📈 **Summary Quality Score**
Shows compression score based on original vs summarized content.

⚡ **Keyboard Shortcut**

```
Ctrl + Enter
```

Run summarization instantly.

---

# 🧮 Algorithms Used

### Frequency Based Summarization

Ranks sentences using **word frequency scoring** after removing stopwords.

### TextRank

A **graph-based ranking algorithm** inspired by PageRank that selects the most important sentences.

# 📦 Installation

Clone the repository

```

```

Install dependencies

```
npm install
```

Run locally

```
npm run dev
```
